# Claude Code Plugins - 제품 문서

## 미션

Claude Code의 기능을 확장하고 개발자 생산성을 극대화하는 플러그인 생태계를 구축합니다.

## 비전

모든 개발자가 자신의 워크플로우에 맞는 Claude Code 플러그인을 쉽게 찾고, 설치하고, 공유할 수 있는 개방형 마켓플레이스를 제공합니다.

---

## 사용자 페르소나

### 주요 사용자

**플러그인 소비자 (Plugin Consumer)**
- Claude Code를 사용하는 개발자
- 생산성 향상 도구를 찾는 팀
- 특정 도메인에 특화된 기능이 필요한 사용자

**플러그인 개발자 (Plugin Developer)**
- 자신의 워크플로우를 플러그인으로 패키징하려는 개발자
- 팀 내부 도구를 공유하려는 조직
- 오픈소스 커뮤니티 기여자

### 사용 시나리오

1. **블로그 작성 자동화**: aStory 플러그인을 통해 8개의 AI 저자 페르소나로 다양한 스타일의 콘텐츠 생성
2. **팀 워크플로우 표준화**: 조직 전체에서 일관된 개발 패턴을 적용
3. **도메인 특화 지원**: 특정 프레임워크나 언어에 최적화된 지원 제공

## 핵심 기능

### 플러그인 구조

| 구성요소 | 설명 | 디렉토리 |
|---------|------|----------|
| Skills | 도메인별 전문 지식 | `.claude/skills/` |
| Agents | 특정 목적에 맞춘 에이전트 | `.claude/agents/` |
| Commands | 슬래시 명령어 | `.claude/commands/` |
| Hooks | 이벤트 기반 자동화 | `.claude/hooks/` |

### 마켓플레이스 기능

- **플러그인 검색**: 카테고리 및 키워드 기반 검색
- **버전 관리**: 시맨틱 버저닝과 변경 로그
- **설치 관리**: 간편한 설치/제거/업데이트
